# CarbonLens

Scan any product and instantly get a carbon footprint breakdown with lifecycle emissions, planetary equivalents, and greener alternatives.

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (database + edge functions)

## Getting started

```sh
npm install
npm run dev
```

## Environment variables

Copy `.env` and fill in your Supabase credentials:

```
VITE_SUPABASE_URL=
VITE_SUPABASE_ANON_KEY=
```
